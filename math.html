VARIABEL A = 6.2

VARIABEL B = 4.6

VARIABEL C = 6.4


Math.round => melakukan/mengembalikan nilai dari pembulatan bilangan terdekat

Hasil pembulatan A = 6
Hasil pembulatan B = 5
Hasil pembulatan C = 6

Math.floor => melakukan/mengembalikan nilai dari pembulatan ke bawah

Hasil pembulatan kebawah A = 6
Hasil pembulatan kebawah B = 4
Hasil pembulatan kebawah C = 6

Math.ceil => melakukan/mengembalikan nilai dari pembulatan ke atas

Hasil pembulatan keatas A = 7
Hasil pembulatan keatas B = 5
Hasil pembulatan keatas C = 7

Math.pow => melakukan perpangkatan dimana x adalah basisnya dan y adalah pangkatnya

Hasil perpangkatan pertama (3 pangkat 3)/(3, 3) = 27
Hasil perpangkatan pertama (5 pangkat 3)/(5, 3) = 125
Hasil perpangkatan pertama (4 pangkat 3)/(4, 3) = 64

Math.sqrt => mengembalikan bentuk akar

Hasil pengembalian bentuk akar 25 = 5
Hasil pengembalian bentuk akar 225 = 15
Hasil pengembalian bentuk akar 64 = 8
